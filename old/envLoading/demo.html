整个页面loading: $.showPageLoading();
取消页面loading: $.hidePageLoading();

某个元素loading: $("").showEnvLoading()
取消某个元素loading: $("").hideEnvLoading()

默认没有右上角的关闭按钮，若需要，需传入onClosed方法(点击关闭按钮可手动取消loading效果)
$.showPageLoading({
     onClosed: function() {
         //do somthing
     }
})
$("").showEnvLoading({
     onClosed: function() {
         //do somthing
     }
})
